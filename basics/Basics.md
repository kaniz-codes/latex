# LaTeX: Basics

## Document Basics
```latex
\begin{document}   % Begin document
\end{document}     % End document
```
## Packages
```latex
\usepackage{amsmath} % Advanced math writing
\usepackage{graphicx} % Insert images
\usepackage{tabularx} % Better table formatting
\usepackage{xcolor} % Add colors to text
\usepackage{hyperref} % Add clickable links and references
\usepackage{geometry} % Set page margins and layout
\usepackage[utf8]{inputenc} % Input encoding
```
## Text Formatting
```latex
\textbf{}      % Bold text
\textit{}      % Italic text
\underline{}   % Underline text
\emph{}        % Emphasized text
```
## Comments
```latex
% This is a comment
```
## Structure Commands
```latex
\section{}       % Section
\subsection{}    % Subsection
\subsubsection{} % Sub-subsection
\title{}         % Title of document
\author{}        % Author name
\date{}          % Date
\maketitle       % Display title
```
## Lists
```latex
\begin{itemize}     % Bullet list
\item Item 1
\end{itemize}

\begin{enumerate}   % Numbered list
\item Item 1
\end{enumerate}
```

## Math Basics
```latex
$ ... $            % Inline math
\[ ... \]          % Display math

^                  % Superscript (x^2)
_                  % Subscript (x_1)
```
## Line Breaks
```latex
\\          % New line (tables/math)
\newline    % Line break in text
```
